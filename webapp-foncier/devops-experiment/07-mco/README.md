## MCO – Support et rapports

Ce dossier regroupe des **modèles** et **exemples** pour la phase de Maintien en Conditions Opérationnelles (MCO).

### Contenu

- `modele_ticket_support.md` : modèle de ticket (incident ou demande).  
- `modele_rapport_mco.md` : modèle de rapport périodique MCO (mensuel, trimestriel).  
- `create_incident_jira_example.py` : exemple de création de ticket d’incident via l’API JIRA Service Management (ou projet JIRA classique).  

Les scripts s’appuient sur les mêmes variables d’environnement que ceux de `06-pilotage/`.
